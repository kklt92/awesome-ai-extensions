# Contributing

Thank you for helping make this list better.

## Adding an extension

Please ensure your pull request follows these guidelines:

- **One extension per pull request.** Multiple additions should be separate PRs.
- The extension must be **currently available** in a browser extension store (Chrome Web Store, Firefox Add-ons, Edge Add-ons) or as an installable open-source project.
- Use the following format: `- [Name](url) - Short description ending with a period.`
- The description should be **concise** (one sentence), **not promotional**, and explain what the extension does.
- Start the description with a lowercase letter (after the dash).
- End the description with a period.
- Add the extension in **alphabetical order** within the appropriate category.
- Check that your suggestion is **not a duplicate**.
- Make sure there are no **dead links** in your PR.

## Removing an extension

Extensions may be removed if they are:

- No longer available in any extension store
- Abandoned with no updates for 12+ months and broken functionality
- Found to contain malware or deceptive behavior

## Updating an extension

If an extension has changed its name, URL, or description, feel free to submit a PR with the correction.

## Creating a new category

If your extension does not fit any existing category, you may propose a new one. A new category should have at least 3 extensions to justify its existence. Open an issue first to discuss.

## Quality standards

We aim to keep this list high quality. Not every AI extension belongs here. We look for:

- Extensions that are **actively maintained**
- Extensions that provide **genuine value** through AI capabilities
- Extensions that are **safe and respect user privacy**

Thank you for your contributions.
